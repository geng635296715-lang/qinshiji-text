# 青筮记应用功能架构

## 1. 产品定位

青筮记是一套以中式玄学咨询为核心的网页应用，当前采用“算法排盘 + AI 命理师咨询 + 用户档案沉淀”的产品结构。

当前主方向：

- 八字命理详解
- 八字合盘分析
- 六爻金钱课
- 梅花易数占卜
- 青筮建议个人中心
- 统一账号、记录与 AI 咨询体系

## 2. 用户端功能结构

### 2.1 首页门面

- 品牌首页展示
- 三大核心入口卡片
- AI 命理师展示卡
- 登录 / 注册入口
- 会员入口保留
- 产品策略说明：VIP 当前限时免费

### 2.2 八字命理

- 公历 / 农历输入
- 天干地支与五行配色展示
- 旺衰 / 喜忌判断
- 神煞结果呈现
- 流年 / 流月分析
- 八字专项解读
- 右侧 AI 命理咨询栏

### 2.3 八字合盘

- 双人八字输入
- 关系类型选择
- 情侣 / 同性情侣 / 同事 / 老板员工 / 合伙人 / 亲人等专项分析
- 合盘图表与旺衰对照
- 喜忌、互补、风险与建议
- 右侧 AI 咨询栏

### 2.4 六爻金钱课

- 一键起卦
- 手动六爻录入
- 老阴 / 老阳 / 少阴 / 少阳录入
- 世应、六亲、六神、伏神、纳甲、月日建等判断
- 寻物 / 感情 / 事业 / 决断专项断语
- AI 咨询联动

### 2.5 梅花易数

- 数字起卦
- 时间起卦
- 外应扩展
- 寻物 / 感情 / 事业 / 决断专项断语
- AI 咨询联动

### 2.6 青筮建议

- 登录后自动读取用户八字档案
- 7 天节奏日历
- 吉日建议
- 吉日分类细化
- 月令 + 黄道吉日 + 个性穿戴建议
- 方向 / 场景建议
- 每日提醒 / 每月提醒配置
- AI 深问

### 2.7 个人中心

- 账号信息
- 八字档案保存
- 最近一次青筮建议摘要
- 历史起卦记录
- AI 咨询记录
- 提醒配置

### 2.8 会员体系

- 当前策略：VIP 内容限时免费
- 保留会员标识和入口
- 未来接入支付宝 / 微信支付
- 后续恢复为付费会员模式

## 3. 后端模块结构

### 3.1 核心接口模块

- `auth`
- `user-center`
- `bazi`
- `divination`
- `qingzhi-advice`
- `ai`

### 3.2 数据存储

- `users`
- `user_sessions`
- `bazi_profiles`
- `user_preferences`
- `divination_records`
- `ai_consultation_records`
- `qingzhi_advice_state`

### 3.3 AI 能力

- DeepSeek 模型接入
- 当前页面结果上下文注入
- 问答历史传递
- 多模块统一 AI 咨询入口
